# Results Folder — Outputs & Provenance

This directory is fully populated with actual, validated predictions, overlays, and vector outputs generated by our fine-tuned SegFormer-B0 model.

## 📁 What is in this folder:

### 1. `training_predictions/`
- Visual overlays (`image1_overlay.png`, `image2_overlay.png`, `image3_overlay.png`) demonstrating the identified grass features with green highlighting on top of the original aerial images.

### 2. `validation_predictions/`
- Spatially accurate GeoJSON files (`image1_prediction.geojson`, `image2_prediction.geojson`, `image3_prediction.geojson`) containing polygonized detections with computed attributes (`area_sqm` and `area_acres`).

### 3. `external_predictions/`
- Predictions on the externally sourced NAIP imagery from Austin, TX (Zilker Park). Includes the visual overlay (`austin_zilker_park_overlay.png`) and the GIS GeoJSON vector output (`austin_prediction.geojson`).

## 📊 Summary of Identified Features:
- **Image 1:** 206 polygons | 18.88 acres turf | 44.8% coverage
- **Image 2:** 108 polygons | 24.62 acres turf | 63.6% coverage
- **Image 3:** 66 polygons | 24.53 acres turf | 80.7% coverage
- **Austin, TX (Generalisation Test):** 167 polygons | 69.87 acres turf | 54.0% coverage

All polygonized detections were morphologically cleaned and simplified to be immediately suitable for downstream GIS workflows in QGIS, ArcGIS, or any coordinate-aware spatial environment.
